## Smart Parking System

### Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: PostgreSQL (Supabase)
- Auth: Phone-based login
- Hosting: Render (backend), Vercel (frontend)

### Features
- Role-based dashboards (User, Manager, Driver)
- Booking history & recent bookings
- Driver assignment & status updates
- Manager revenue & stats
